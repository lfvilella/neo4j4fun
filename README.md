# neo4j4fun
Learning about [neo4j](https://neo4j.com/) (graph db) and testing cypher query language.

## Dependecies

- Docker
- docker-compose

## Running locally with Docker

    $ docker-compose up

Go to `http://localhost:7474` and have fun.
